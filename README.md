# motors

motors module is a Python user interface for to motion controlers

Controlers avaible :
RSAI
NewFocus
SmartAct
Newport
TMCM
pololu
Arduino


test on windows 32 bits only

Requirements
python 3.x
Numpy
PyQt5
qdarkstyle (https://github.com/ColinDuquesnoy/QDarkStyleSheet.git) pip install qdarkstyle
