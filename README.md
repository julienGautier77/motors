# motors

motors module is a Python user interface based on PyQT for to control motion controlers

Controlers avaible :

-RSAI (64 bits Dll)

-NewFocus

-SmartAct

-Newport

-TMCM

-pololu

-Arduino

-APT (Thorlabs)


test on windows 64 bits 

Requirements
python 3.x
Numpy
PyQt6
qdarkstyle (https://github.com/ColinDuquesnoy/QDarkStyleSheet.git) pip install qdarkstyle
